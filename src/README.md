# Java Core: OOP Fundamentals

This repository is a structured collection of Java exercises, focused on core Object-Oriented Programming principles. It serves as a personal knowledge base for my journey as a **Fachinformatiker für Anwendungsentwicklung (FIAE)**.

## 📂 Repository Structure

- **encapsulation**: Demonstrates data protection using `private` fields and validation logic.
  - Examples: `BankCard`, `Scooter`, `Wallet`, `FlightTicket`.
- **inheritance**: Showcases class hierarchies and code reuse.
  - Examples: `Employee` hierarchy (`Manager`, `Developer`, `Intern`).
- **abstraction**: Working with abstract concepts and common interfaces.
  - Examples: `Shape` (Circle, Triangle), `Workout` (Yoga, Running).

## 🚀 Key Learning Objectives
- Understanding access modifiers (`private`, `public`, `protected`).
- Implementing business logic inside setters and constructors.
- Creating maintainable and scalable class structures.
